# Attestia Desktop

Financial intent verification for Windows — WinUI 3 app and .NET SDK for blockchain attestation and reconciliation.

## Key Features

- **Intent verification** — cryptographic proof that financial actions match stated intent
- **Blockchain attestation** — tamper-evident record of financial operations
- **Reconciliation engine** — automated matching of intent to outcome
- **WinUI 3 desktop app** — native Windows experience
- **.NET SDK** — Core, Client, and Sidecar packages for integration

## NuGet Packages

```bash
dotnet add package Attestia.Core
dotnet add package Attestia.Client
dotnet add package Attestia.Sidecar
```
